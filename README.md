## WonderTrip - Website Pariwisata Indonesia

Website pariwisata yang menampilkan destinasi wisata populer di tiga kota: Surabaya, Makassar, dan Semarang.

## Fitur

- Menampilkan destinasi wisata dari 3 kota
- Sistem pemesanan tiket
- Pencarian destinasi
- Admin panel untuk mengelola data
- Responsive design

## Instalasi

1. Upload semua file ke web server
2. Buat database MySQL
3. Import file SQL (jika ada)
4. Konfigurasi koneksi database di `connect.php`
5. Akses website melalui browser
